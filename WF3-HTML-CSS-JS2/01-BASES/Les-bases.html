Les balises DIV

Les balises de type DIV est une balise neutre et block

Les balises Block agissent sur un axe vertical et donc génère un retour a la ligne .
Je pourrais toute-fois changer leur comportement avec du CSS.

Les balises Inline elles , agissent sur un axe horizontal et donc ne genere pas de retour a la ligne .
Je pourrais toute-fois chengaer leur comportement avec du CSS.

Les formulaires :

  • Les balises Form permet de créer le formulaire.
  • Name: Permet de donner un nom au formulaire.
  • Action: Cet attribut est OBLIGATOIRE et permet de renseigner le script/ La page vers la quelle l'utilisateur sera dirigé a l'envoi du script. Cette page va recevoir l'ensemble des données saisies par mon utilisateur
  • Method Permet de préciser quelle méthode [GET | POST] doit être utilisé par le navigateur pour transmettre les données saisies par l'utilisateur vers la page ACTION

Créer un lien en html

  • la balise "a" permet de créer un lien vers une autre ressource . (Images , Vidéos, PDF, ZIP etc...)

  • l'Aattribut href: permet de définir le chemin vers le fichier a lier

  • L'attribut target : prend plusieurs valeurs : "_blank" pour ouvrir le lien dans une nouvelle fenetre par exemple

  • L'attribut title : permet de donner un titre au lien . Ce titre sera affiché dans une petite bulle au survol du line par la souris
